Finish a lesson in the book

Experiment. Change values. Break things. Fix them.

Learning to code is not about being perfect.
It’s about trying, thinking, and improving.

## A Small Note for Parents

If your child gets stuck, that’s completely normal.
Instead of giving the answer right away, try asking:

“What do you think will happen if we change this?”
“Why do you think the computer is showing that message?”

Coding builds confidence when children discover answers themselves.
